# IBM-HR-Analytics
This project focuses on analyzing employee attrition using the IBM HR dataset. The goal is to identify key factors influencing employee turnover and provide actionable insights using SQL for data preprocessing and Power BI for visualization.
## ️ Tools & Technologies


SQL (PostgreSQL/MySQL) – Data preprocessing & analysis 

#Power BI – Dashboard & data visualization 

Excel/CSV – Data source 

## Dataset

The dataset contains employee-related information such as: 

Age 

Department 

Job Role 

Monthly Income 

Attrition (Yes/No) 

Work-Life Balance 

Job Satisfaction 

Overtime 

## Key Analysis Performed


Percentage of employees leaving vs staying 

Attrition trends across departments 

Attrition by job role 

Impact of overtime on attrition 

## Dashboard Features


KPI Cards (Total Employees, Attrition Rate) 

Department-wise Attrition Charts 

Salary Distribution Visuals 

Interactive Filters (Department, gender,overtime) 

## Conclusion

This project demonstrates how data analytics can help organizations identify attrition patterns and improve employee retention strategies.
